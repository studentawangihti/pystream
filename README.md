Local high-quality video streaming app (VDO.Ninja clone) using WebRTC & Flask.

## Fitur
- 4K 60FPS Support
- Low Latency (P2P)
- Hybrid UI (Mobile & Desktop Friendly)
- OBS Clean Feed support

## Cara Pakai
1. Install dependencies: `pip install -r requirements.txt`
2. Jalankan server: `python app.py`
3. Buka browser di HP (Broadcaster) dan Laptop (Viewer).
4. Pastikan menggunakan HTTPS dan satu jaringan Wi-Fi.
